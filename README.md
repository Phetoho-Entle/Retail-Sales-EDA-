Retail Sales Exploratory Data Analysis
Project Overview

Brief description of the project and dataset.

Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

Data Cleaning
- Checked for missing values
- Checked for duplicates
- Converted date column to datetime format

Key Findings
- Electronics generated the highest revenue.
- Customers aged 37 had the highest average spending.
- Customers aged 39 had the lowest average spending.
- Sales peaked the highest in May.
- Female customers contributed a larger share of total sales.

Recommendations
- Increase inventory and marketing efforts for high-performing categories.
- Target high-spending customer segments with tailored promotions.
- Prepare inventory and staffing levels ahead of peak sales periods.
- Investigate strategies to improve performance in lower-selling categories.
